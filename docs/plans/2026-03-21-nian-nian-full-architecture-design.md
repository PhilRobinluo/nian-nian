## 核心架构：输入 → 整理 → 输出

```
输入层（4种模式）        →    整理层（AI 引擎）      →    输出层（多种形式）
─────────────────          ────────────────          ────────────────
🎙️ 实时语音对话             🧠 资料整理中心            📖 回忆录预览
   - 预设问题库（按人生阶段）   - 素材去重合并            📄 PDF 导出
   - 语音消息（按住说话）       - 时间线自动排序          🖼️ 长图/海报
   - AI 实时文字回复           - 智能分章节             🔗 分享链接
   - 智能追问                 - 润色成第一人称叙述
                              - 关联照片
📁 导入录音
   - 上传音频文件
   - ASR 转文字（Web Speech API / Whisper）
   - AI 提取关键故事

📄 导入资料
   - 粘贴文字（日记/书信/微信记录）
   - 上传照片 + 说明
   - 批量导入

✏️ 资料管理
   - 查看所有已收集素材
   - 编辑/补充/删除
   - 手动调整章节
```

## 页面架构

```
/                              → 首页（档案列表 + 创建入口）
/profile/new                   → 创建新档案
/profile/[id]                  → 档案主页（功能入口中心）
│
├── /profile/[id]/chat         → 🎙️ 实时语音对话（预设问题 + 自由聊）
├── /profile/[id]/import       → 📁📄 导入中心（录音 / 文字 / 照片）
├── /profile/[id]/materials    → ✏️ 资料管理（所有素材列表 + 编辑）
├── /profile/[id]/organize     → 🧠 AI 整理（一键整理 → 预览章节）
└── /profile/[id]/memoir       → 📖 回忆录（阅读 + 导出 + 分享）
```

## 功能模块详细设计

### 模块 1：实时语音对话 `/chat`

**交互方式**：语音消息模式（按住说话 → 松开转文字 → AI 文字回复）

**预设问题库**（按人生阶段分 6 大类，每类 8-10 个问题）：

1. **童年记忆**（0-12岁）
   - 您小时候住在哪里？家是什么样子的？
   - 小时候最喜欢玩什么？有哪些好朋友？
   - 上学时印象最深的老师是谁？
   - 家里有几口人？爸妈是做什么的？
   - 小时候最喜欢吃什么？妈妈做的什么菜最好吃？
   - 过年的时候是怎么过的？有什么特别的习俗？
   - 小时候调皮捣蛋做过什么事？
   - 那时候最大的梦想是什么？

2. **少年时光**（13-18岁）
   - 在哪里上的中学？学校大不大？
   - 那时候最擅长什么科目？
   - 有没有特别要好的同学？现在还联系吗？
   - 第一次离开家是什么时候？去哪里了？
   - 那个年代最流行什么？
   - 有没有偷偷喜欢过谁？
   - 当时最想做什么工作？
   - 人生中第一个转折点是什么？

3. **青年奋斗**（19-30岁）
   - 第一份工作是什么？怎么找到的？
   - 第一次领工资是什么感觉？花在哪里了？
   - 怎么认识您爱人的？第一次见面什么印象？
   - 结婚那天是什么样的？
   - 最难的一段时间是什么时候？怎么挺过来的？
   - 那时候一个月挣多少钱？够花吗？
   - 有没有后悔过的选择？
   - 最自豪的一件事是什么？

4. **中年岁月**（31-50岁）
   - 孩子出生的时候是什么感觉？
   - 在工作上最大的成就是什么？
   - 有没有特别感激的贵人？
   - 生活中最大的变化是什么？
   - 那些年最操心的事是什么？
   - 有没有搬过家？从哪搬到哪？
   - 教育孩子上有什么心得？
   - 夫妻之间吵过最大的一次架是因为什么？

5. **晚年感悟**（50+岁）
   - 退休后每天都做些什么？
   - 回头看这一辈子，最满意的是什么？
   - 有没有什么遗憾的事？
   - 最想对子女说的一句话是什么？
   - 现在最大的心愿是什么？
   - 觉得这辈子最幸福的时刻是哪一刻？
   - 对年轻人有什么建议？
   - 如果重来一次，会做什么不同的选择？

6. **自由话题**
   - AI 根据前面的对话自动生成追问
   - 用户也可以自己输入任何问题

**对话界面设计**：
- 顶部：当前话题标签 + 切换话题
- 中间：聊天气泡（左 AI 右用户）
- 底部：预设问题快捷按钮（可滑动）+ 语音按钮 + 文字输入框
- 语音按钮：按住录音，松开转文字发送

### 模块 2：导入中心 `/import`

三个 Tab 页：

**Tab 1 - 导入录音**
- 上传音频文件（mp3/m4a/wav）
- 显示转写进度
- 转写完成后预览文字，可编辑修正
- 确认后存入素材库

**Tab 2 - 导入文字**
- 大文本框，粘贴任何文字内容
- 来源标签（日记/书信/聊天记录/其他）
- 时间标注（大约什么年代的内容）
- 确认导入

**Tab 3 - 导入照片**
- 上传照片（支持多张）
- 每张照片配说明文字
- 标注时间和人物
- 确认导入

### 模块 3：资料管理 `/materials`

- 所有素材的时间线视图
- 按来源分类筛选（对话/录音/文字/照片）
- 每条素材可编辑/删除/标注重要
- 素材总量统计
- "一键整理"按钮 → 跳转到整理页

### 模块 4：AI 整理 `/organize`

- 显示当前所有素材概览
- 点击"开始整理" → AI 处理动画
- 整理完成后展示：
  - 自动生成的章节列表
  - 每个章节的内容预览
  - 时间线可视化
- 可手动调整章节顺序/合并/拆分
- 确认后生成回忆录

### 模块 5：回忆录 `/memoir`

- 封面页（老人姓名 + 照片 + 年份 + 书名）
- 目录页
- 各章节内容（第一人称、图文混排）
- 底部操作：导出 PDF / 生成长图 / 复制分享链接

## UI 设计原则

- **操作者是年轻人**（25-50岁），UI 现代精致但温暖
- 配色：暖棕色系保留，但更精致（#8B6914 金棕点缀，主体用中性暖灰）
- 移动端优先，max-w-md 居中
- 档案主页用大图标卡片展示 4 个功能入口
- 对话页底部预设问题用横向滑动 pill 按钮

## 技术方案

- **前端**：Next.js 16 + shadcn/ui + Tailwind CSS 4
- **语音转文字**：Web Speech API（MVP）→ Whisper API（升级）
- **AI 引擎**：Vercel AI SDK + AI Gateway
- **数据**：MVP 用 localStorage，后续接 Supabase
- **部署**：Vercel
